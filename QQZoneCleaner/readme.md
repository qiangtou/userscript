###代码说明

过滤动作是在页面有滚动的时候发生

有点动手能力（会js）的可以自己加一些简单的配置。
代码开头有三个比较关键的数组。
通过指定的class数组，id数组或者关键字来过滤你不想看到的一切内容。

转载也去掉了~~~

####classArr 
里面放你要删掉的元素的class

####idArr 
里面放你要删掉的元素的id

####filterstr
在filterstr这个数组里面添加你要过滤的关键字，脚本会扫描当前面所有的用户昵称，以及用户发的说说，状态，来进行过滤。
